
# Random Password Generator

## What it does
Creates strong, random passwords instantly.

## How it works
1. Choose password length (4-20)
2. Click "Generate"
3. Gets random password
4. Click "Copy" to use it

## Password Includes
- Uppercase letters (A-Z)
- Lowercase letters (a-z)
- Numbers (0-9)
- Special characters (!@#$%^&*)
